# Expense Tracker

**Автор:** Сомов Дмитрий Владиславович

## Описание

Expense Tracker — приложение для отслеживания личных расходов с графическим интерфейсом на Python (с использованием библиотеки Tkinter).

## Основные функции

* Добавление расходов с указанием суммы, категории и даты.
* Отображение всех расходов в виде таблицы.
* Фильтрация расходов:
  * по категории;
  * по периоду дат;
  * комбинированная фильтрация.
* Подсчёт общей суммы расходов (в т. ч. по отфильтрованным данным).
* Сохранение и загрузка данных в формате JSON.
* Валидация ввода данных (сумма, дата, выбор категории).

## Требования

* Python 3.6 или выше.
* Стандартные библиотеки: `tkinter`, `json`, `datetime`, `os`.

## Инструкция по запуску

1. **Создайте папку для проекта:**
   ```bash
   mkdir expense-tracker
   cd expense-tracker

Примеры использования: 

Пример 1. Добавление расхода
В поле «Сумма» введите число (например, 1500).

В выпадающем списке «Категория» выберите одну из категорий (например, «Еда»).

В поле «Дата (ГГГГ‑ММ‑ДД)» введите дату в правильном формате (например, 2024-04-25).

Нажмите кнопку «Добавить расход».

Новая запись появится в таблице внизу окна, а общая сумма обновится.

Пример 2. Фильтрация по категории
В фильтре «Категория» выберите нужную категорию (например, «Транспорт»).

Нажмите кнопку «Применить фильтр».

Таблица обновится — в ней останутся только расходы выбранной категории. Общая сумма пересчитается для отфильтрованных данных.

Пример 3. Фильтрация по периоду дат
В поле фильтра «С» введите начальную дату периода (например, 2024-04-01).

В поле фильтра «По» введите конечную дату периода (например, 2024-04-30).

Нажмите «Применить фильтр».

В таблице останутся только расходы за указанный период.
